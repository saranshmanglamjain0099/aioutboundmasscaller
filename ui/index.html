🔧 Quick Start Guide

  1. Run supabase_schema.sql in Supabase Dashboard → SQL Editor
  2. Go to ⚙️ Settings → fill in LiveKit, Gemini, and Vobiz credentials → Save each
  3. Click ⚡ Create SIP Trunk in the Vobiz section
  4. Go to ✏️ AI Prompt → customize for your business → Save
  5. Go to 🤖 Agents → create an agent profile
  6. Go to 📞 Single Call → test with your own number
